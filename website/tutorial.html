Tutorial

  • $ai [your message here] - Use this to talk with the bot!
  • $setprefix [Your custom prefix here] - Allows you to change character in front of message (Default is $)
  • $setkey [Your key pasted from the OpenAI website] - Allows you to use your own key for the bot, letting you use it for longer. (HIGHLY RECOMMENDED)
  • $help - When you need a reminder of the commands for the bot
